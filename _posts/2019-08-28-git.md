git是代码管理的工具，本文记录一些git的使用方法

#### git项目迁移

1. ```
git clone --bare <旧git地址>
```
2. ```
git push --mirror <新git地址>
```

我用上面方法把gitlab的项目迁移到了github，从命令上看，各种git相关工具应该都能使用这种方式,关于两个命令的说明参考[git文档][]

#### 本地仓库指定远程仓库

```
git remote add origin
```
